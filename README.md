# Kurzbeschreibung des Projekts
In diesem Projekt wurde eine Webanwendung entwickelt, die in Echtzeit sowie über einen zeitlichen Verlauf darstellt, wie viele Menschen sich aktuell an bestimmten Orten in Luzern aufhalten. Die Daten werden regelmaessig über eine externe API abgerufen, mit PHP transformiert und in einer eigenen MySQL-Datenbank gespeichert. Auf der Webseite werden die aggregierten Daten visuell aufbereitet und in Form von Kennzahlen, Diagrammen sowie einer Stadtkarte uebersichtlich dargestellt.

⸻

# Learnings
	•	Umsetzung eines vollständigen ETL-Prozesses (Extract, Transform, Load) mit PHP
	•	Arbeiten mit zeitabhaengigen Live-Daten aus einer API
	•	Speicherung und Abfrage von Daten in einer relationalen Datenbank (MySQL)
	•	Strukturierte Zusammenarbeit in einem gemeinsamen GitHub-Repository

⸻

# Schwierigkeiten
	•	Korrektes Transformieren und Speichern der API-Daten in der eigenen Datenbank
	•	Aufbau einer stabilen Datenbankstruktur sowie Fehleranalyse beim Schreiben der Daten
	•	Weiterverarbeitung und Darstellung der gespeicherten Daten im Frontend
	•	Durcheinander auf GitHub. Verschiedene Zwischenstände gleichzeitig hochgeladen.
